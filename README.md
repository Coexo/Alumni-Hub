# 🎓 Alumni Hub – *Hackathon Track Winner*

An AI-powered alumni engagement platform designed to connect students and alumni through real-time communication, intelligent matching, and career-oriented features.  
Built with **ReactJS**, **Node.js**, **Express.js**, **Python**, and **MongoDB**.

---

## 🛠️ Tech Stack

- **Frontend:** ReactJS, Material UI  
- **Backend:** Node.js, Express.js, Python (Flask)  
- **Database:** MongoDB  
- **Other Tools:** Socket.IO, AI/ML (for skill matching)

---

## ✨ Features

- 🏠 **Homepage Dashboard** – Clean overview of platform features and updates  
- 💬 **Discussion Forum** – Post questions, doubts, and mentorship topics with threaded replies  
- 📹 **Real-time Chat & Video Calls** – One-on-one or group communication between alumni and students  
- 💼 **Jobs & Internship Board** – Browse and apply to opportunities posted by alumni or institutions  
- 📅 **Events Page** – View and register for upcoming webinars, reunions, and tech talks  
- 📚 **Courses & Resources Section** – Curated learning paths and recommended skill-based content  
- 👤 **Profile Page** – Dynamic profiles displaying education, experience, skills, and interaction history  
- 🧠 **AI-Powered Skill Matching** – Recommends relevant alumni based on student skill profiles using Python logic
